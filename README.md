Public Transport Data Analysis (2019–2022)

Project Overview



This project presents a comparative analysis of public transport usage using two datasets from 2019 and 2022. The aim is to explore changes in travel behaviour, system usage, and passenger patterns over time by analysing both aggregated system-level data and detailed journey-level data.



The analysis highlights how public transport usage evolved between the two periods and provides insights into ridership trends, travel behaviour, and operational patterns.



Datasets

2019 Dataset



Daily aggregated public transport data



Covers:



* Bus, tram, and metro usage



* Peak and off-peak ridership



* Daily revenue figures



* Each row represents one day of system-wide activity



2022 Dataset



Journey-level transport data



Includes:



* Date and time
* Transport mode
* Distance travelled
* Journey duration
* Fare price



Each row represents an individual trip



Objectives



* Compare public transport usage patterns between 2019 and 2022



* Analyse changes in passenger behaviour over time



* Identify trends in ridership, demand, and travel characteristics



* Contrast system-level aggregates with detailed trip-level data



* Apply statistical and exploratory data analysis techniques





&nbsp;Tools \& Technologies



* Python



* Pandas – data manipulation



* NumPy – numerical analysis



* Matplotlib / Seaborn – data visualisation



* Jupyter Notebook – analysis and reporting





Key Analysis Performed



* Data cleaning and preprocessing



* Exploratory Data Analysis (EDA)



* Trend comparison across years



* Peak vs off-peak usage analysis



* Journey distance and duration analysis



* Visual comparison of 2019 vs 2022 travel patterns



Key Insights



* Clear differences in travel behaviour between 2019 and 2022



* Increased granularity in 2022 allows deeper passenger-level insights



* Observable changes in demand patterns and journey characteristics



* Evidence of behavioural shifts in public transport usage over time
